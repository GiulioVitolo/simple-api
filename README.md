### HOW TO RUN ###
1) Clone the repository
2) Open the terminal
3) Navigate to the root folder (= '/simple-api', where server.ts is located)
4) Run 'npm install' to install all the necessary dependencies
5) Run 'npm run dev' to run the (locally hosted) server
